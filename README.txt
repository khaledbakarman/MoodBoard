Required Mood Images

Please add the following images to this directory:

1. happy.jpg     - A happy, smiling person or sunny scene
2. sad.jpg       - A rainy day or melancholic scene
3. excited.jpg   - Someone celebrating or jumping with joy
4. relaxed.jpg   - A peaceful nature scene or meditation setting
5. energetic.jpg - Someone exercising or being active
6. tired.jpg     - A yawning person or sunset scene
7. inspired.jpg  - Creative workspace or artistic scene
8. grateful.jpg  - Family gathering or peaceful moment

Image Guidelines:
- Format: JPG/JPEG
- Size: 800x600 pixels recommended
- File size: Keep under 500KB each
- Name: Must match exactly as listed above (all lowercase)

You can find free images from:
1. Unsplash (https://unsplash.com)
2. Pexels (https://pexels.com)
3. Pixabay (https://pixabay.com)

For testing, you can use any appropriate images but rename them according to the mood names above.

Note: In a production environment, you would implement proper image upload functionality and storage.
